# finanzas-1.0

# 💰 Finanzas Ágiles

**Aplicación web sencilla de planeación financiera** diseñada para personas con ingresos restringidos.

---

## ✨ Características

- Dashboard claro con resumen financiero
- Registro fácil de gastos
- Simulador de metas financieras
- Recomendaciones prácticas de ahorro e inversión
- Interfaz limpia y fácil de usar

---

## 🚀 Cómo Ejecutar

```bash
# 1. Clonar repositorio
git clone https://github.com/tuusuario/finanzas-faciles.git
cd finanzas-faciles

# 2. Instalar dependencias
pip install -r requirements.txt

# 3. Ejecutar
streamlit run app.py
